Html 5 Concepts

HTML 5 is the fifth and current version of HTML. It has improved the markup available for documents and has introduced application programming interfaces(API) and Document Object Model(DOM). It has introduced various new features like drag and drop, geo-location services
  1. It supports audio and video controls tags.
  2. Allows JavaScript to run in the background.
  3. Vector graphics are additionally an integral part of HTML5 like SVG and Canvas.
  4. HTML5 language is more mobile-friendly.
  5. One can track the GeoLocation of a user easily by using JS GeoLocation API
